# SQL  — Northwind Database

Practiced SQL on the Northwind sample database.

## Concepts Used
SELECT, FROM, WHERE, ORDER BY, LIMIT, DISTINCT,
AND, OR, NOT, IN, BETWEEN, LIKE,
IS NULL, IS NOT NULL, COALESCE

## Files
- `northwind_setup.sql` — sets up the database
- `business_questions.sql` — 10 business queries with explanations

## Questions Answered
1. Who are our customers and where are they from?
2. Which countries do our customers come from?
3. Which products are discontinued?
4. Which products are priced between $20 and $100?
5. Which orders have NOT been shipped yet?
6. Show the 5 most expensive products we sell.
7. Which customers have no email address on record?
8. Find all products in the 'Beverages' or 'Sauces' category.
9. Which employees are based in Seattle or Redmond?
10. Show each order's shipping fee — display 'No fee' if it is NULL or zero.
